# 📊 Sales Analysis & Dashboard Project

## 📌 Project Overview

This project focuses on analyzing sales data and building an interactive dashboard to gain business insights. The goal is to understand sales performance across categories, regions, and time.

## 🎯 Objectives

* Analyze total sales performance
* Identify top-performing products
* Understand regional sales distribution
* Track monthly sales trends
* Build an interactive dashboard

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Streamlit

## 📂 Dataset

* Superstore Sales Dataset (`train.csv`)
* Contains information about:

  * Order Date
  * Region
  * Category
  * Product Name
  * Sales
  * Customer ID

## 📊 Key Analysis Performed

### ✔️ Sales Analysis

* Total Sales Calculation
* Category-wise Sales
* Region-wise Sales

### ✔️ Time-Based Analysis

* Monthly Sales Trends

### ✔️ Product Analysis

* Top 10 Best-Selling Products

## 📈 Visualizations

* Bar Chart (Sales by Category)
* Pie Chart (Sales by Region)
* Line Chart (Monthly Sales Trend)
* Horizontal Bar Chart (Top Products)

## 🚀 Dashboard Features

* Interactive filters (Region & Category)
* Key metrics display
* Dynamic charts
* Real-time data updates

## ▶️ How to Run the Project

1. Install required libraries:

   ```bash
   pip install pandas matplotlib streamlit
   ```

2. Run the dashboard:

   ```bash
   streamlit run app.py
   ```
## 💡 Key Insights

* Certain categories contribute more to overall sales
* Sales vary significantly across regions
* Monthly trends help identify peak business periods
* A few products generate major revenue

## 📌 Conclusion

This project demonstrates how data analysis and visualization can help businesses make informed decisions and improve performance.
